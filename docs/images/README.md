README 里对应的图片就会自动显示（把 README 中该图的 HTML 注释去掉即可）。

建议截图与文件名：

| 文件名 | 页面 | 脱敏提醒 |
| --- | --- | --- |
| `overview.png`   | 总览 / 路由（协议互转）   | —— |
| `providers.png`  | Provider 列表 / 卡片      | **打码 OAuth 账号邮箱**、密钥（UI 默认已掩码，仍需复核） |

| `public-url.png` | 公网访问（Cloudflare）    | **打码自有域名 / Tunnel Token** |
| `usage.png`      | 用量 / 日志                | 打码客户端真实 IP、账号邮箱 |
